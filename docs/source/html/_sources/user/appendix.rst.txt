..     |                     |                                                    |
..     |                     | ``'thickness',[],... % thickness [mm]``            |
..     |                     |                                                    |
..     |                     | ``% of single ply``                                |
..     |                     |                                                    |
..     |                     | ``'angle',[],... % ply angle``                     |
..     |                     |                                                    |
..     |                     | ``'nPlies',[]); % number of plies``                |

..     | ``indices``         | Indices of stack                                   |

..     |                     | [in board station, out board station, 1\ :sup:`st` |
